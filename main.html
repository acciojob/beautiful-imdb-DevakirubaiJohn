Beautiful IMDb

People are bored with the original movie detail page of IMDb. Your task is to update a section for them and make it more beautiful. Below are the mandatory requirements for the movie detail page. Feel free to add more styles to make it more appealing.

  1. The poster for the movie:
    1. Must be inside an “img” tag with a class of “moviePic”.
    2. Should have CSS properties: height, width, and opacity.
  2. The description of the movie:
    1. Must be inside a “p” tag with a class of “movieDetail”.
    2. Should be in the center of the page using CSS properties: position, top, width, text-align, and font-size.

Note: Don’t use inline CSS, instead use class selectors for CSS,

Element Description Class CSS Properties
img Movie Poster moviePic height, width, opacity
p Movie Description movieDetail position, top, width, text-align, font-size
